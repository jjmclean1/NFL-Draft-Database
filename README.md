# Draft Stock

**Draft Stock** is a scouting application designed for NFL Draft analysis.

It provides users with film cut-ups, statistical matrices, athletic profiles, and advanced comparison tools to evaluate prospects.

## Key Features
                                                            
* **Complete Prospect Profiles:** Deep-dive views including Combine/Pro Day testing measurables, historical game logs, and segmented visual bars for scout grading.
* **The Matrix (Dynamic Player Comparisons):** A scrollable data table with a frozen left column. It allows users to stack prospects side by side and compare advanced metrics (YPRR, ADOT, Contested Catch %, etc.).
* **Intelligent Heat-Mapping:** Stats are automatically color-coded (red, green, yellow) based on standard draft-class baseline averages, highlighting a player's strengths and weaknesses.
